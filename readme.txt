This is a simple plugin that lets Gravity Forms users disable form fileds by adding a CSS class of "disabled".

== Description ==

No lengthy description needed. Simply install the plugin and add the class "disabled" (no quotes) to a form field. This is done within the Form Editor under the Advanced tab of the field.

== Installation ==

This section describes how to install the plugin and get it working.

<h2>Automatically</h2>

1. Search for Disable Gravity Forms Fields in the Add New Plugin section of the WordPress admin
2. Install & Activate

<h2>Manually</h2>

1. Download the zip file and upload `disable-gf-fields` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I use this plugin? =

Once you've installed and activated the plugin, just add a class named "disabled" to your field. That's all.

= Do you offer support for this plugin? =

This plugin is really as simple as it gets. Therefore, it's unlikely you'll need support.
